## HTML代码

在HTML中嵌入一下代码(基于JEKYLL)

```html
<div class="extensions extensions--video">
  <iframe src="//player.bilibili.com/player.html?aid={{ include.id }}&page=1&high_quality=1&danmaku=0"
    frameborder="no" scrolling="no" allowfullscreen="true">
  </iframe>
</div>
```

```css
.extensions {
    margin: 0 auto;
    display: block;
}

.extensions--video{
  position: relative;
  width: 100%;
  padding: 0;
  margin-bottom: 5px;
  border-radius: 10px;
  box-shadow: 0 2px 4px 1px rgb(28, 29, 77);
  & > iframe {
    position: absolute;
    top: 0;
    left: 0;
    width: 100%;
    height: 100%;
  }
}
  
.extensions--video {
  padding-top: percentage(315 / 560);
  box-shadow: 0 2px 4px 1px rgb(28, 29, 77);
  border-radius: 10px;
}
```

## Bilibil 网页参数

| key          | 说明                                                    |
| ------------ | ------------------------------------------------------- |
| aid          | 视频ID 就是B站的avxxxx 后面的数字                       |
| cid          | 应该是客户端id,经测试, 这个字段不填也没关系             |
| page         | 第几个视频, 起始下标为 1 (默认值也是为1)                |
| as_wide      | 是否宽屏 1: 宽屏, 0: 小屏                               |
| high_quality | high_quality=1 最高1080p: <br />high_quality=0 最低360p |
| danmaku      | 是否开启弹幕 1:开启(默认), 0:关闭                       |
